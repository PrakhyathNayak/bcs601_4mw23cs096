# BCS601-USN096

## Project Title
Number Operations Web Application using Python

## Introduction
This project is a simple web application developed using Python and Flask. It performs operations such as Fibonacci series generation, prime number checking, palindrome verification, and Armstrong number checking.

## Features
- Fibonacci series generation
- Prime number checking
- Palindrome checking
- Armstrong number verification
- Web-based output using Flask

## Technologies Used
- Python
- Flask
- GitHub
- Render
